2016-10-21

1 - WEB е§ӢзҘ– HTTP

е…Ёз§°пјҡи¶…ж–Үжң¬дј иҫ“еҚҸи®®(HyperText Transfer Protocol)

дјҙйҡҸзқҖи®Ўз®—жңәзҪ‘з»ңе’ҢжөҸи§ҲеҷЁзҡ„иҜһз”ҹпјҢHTTP1.0д№ҹйҡҸд№ӢиҖҢжқҘпјҢеӨ„дәҺи®Ўз®—жңәзҪ‘з»ңдёӯзҡ„еә”з”ЁеұӮпјҢHTTPжҳҜе»әз«ӢеңЁTCPеҚҸи®®д№ӢдёҠпјҢжүҖд»ҘHTTPеҚҸи®®зҡ„з“¶йўҲеҸҠе…¶дјҳеҢ–жҠҖе·§йғҪжҳҜеҹәдәҺTCPеҚҸи®®жң¬иә«зҡ„зү№жҖ§пјҢдҫӢеҰӮtcpе»әз«ӢиҝһжҺҘзҡ„3ж¬ЎжҸЎжүӢе’Ңж–ӯејҖиҝһжҺҘзҡ„4ж¬ЎжҢҘжүӢд»ҘеҸҠжҜҸж¬Ўе»әз«ӢиҝһжҺҘеёҰжқҘзҡ„RTTе»¶иҝҹж—¶й—ҙгҖӮ

2 - HTTP дёҺзҺ°д»ЈеҢ–жөҸи§ҲеҷЁ

ж—©еңЁHTTPе»әз«Ӣд№ӢеҲқпјҢдё»иҰҒе°ұжҳҜдёәдәҶе°Ҷи¶…ж–Үжң¬ж Үи®°иҜӯиЁҖ(HTML)ж–ҮжЎЈд»ҺWebжңҚеҠЎеҷЁдј йҖҒеҲ°е®ўжҲ·з«Ҝзҡ„жөҸи§ҲеҷЁгҖӮд№ҹжҳҜиҜҙеҜ№дәҺеүҚз«ҜжқҘиҜҙпјҢжҲ‘д»¬жүҖеҶҷзҡ„HTMLйЎөйқўе°ҶиҰҒж”ҫеңЁжҲ‘д»¬зҡ„webжңҚеҠЎеҷЁдёҠпјҢз”ЁжҲ·з«ҜйҖҡиҝҮжөҸи§ҲеҷЁи®ҝй—®urlең°еқҖжқҘиҺ·еҸ–зҪ‘йЎөзҡ„жҳҫзӨәеҶ…е®№пјҢдҪҶжҳҜеҲ°дәҶWEB2.0д»ҘжқҘпјҢжҲ‘д»¬зҡ„йЎөйқўеҸҳеҫ—еӨҚжқӮпјҢдёҚд»…д»…еҚ•зәҜзҡ„жҳҜдёҖдәӣз®ҖеҚ•зҡ„ж–Үеӯ—е’ҢеӣҫзүҮпјҢеҗҢж—¶жҲ‘д»¬зҡ„HTMLйЎөйқўжңүдәҶCSSпјҢJavascriptпјҢжқҘдё°еҜҢжҲ‘д»¬зҡ„йЎөйқўеұ•зӨәпјҢеҪ“ajaxзҡ„еҮәзҺ°пјҢжҲ‘д»¬еҸҲеӨҡдәҶдёҖз§Қеҗ‘жңҚеҠЎеҷЁз«ҜиҺ·еҸ–ж•°жҚ®зҡ„ж–№жі•пјҢиҝҷдәӣе…¶е®һйғҪжҳҜеҹәдәҺHTTPеҚҸи®®зҡ„гҖӮеҗҢж ·еҲ°дәҶз§»еҠЁдә’иҒ”зҪ‘ж—¶д»ЈпјҢжҲ‘д»¬йЎөйқўеҸҜд»Ҙи·‘еңЁжүӢжңәз«ҜжөҸи§ҲеҷЁйҮҢйқўпјҢдҪҶжҳҜе’ҢPCзӣёжҜ”пјҢжүӢжңәз«Ҝзҡ„зҪ‘з»ңжғ…еҶөжӣҙеҠ еӨҚжқӮпјҢиҝҷдҪҝеҫ—жҲ‘д»¬ејҖе§ӢдәҶдёҚеҫ—дёҚеҜ№HTTPиҝӣиЎҢж·ұе…ҘзҗҶи§Је№¶дёҚж–ӯдјҳеҢ–иҝҮзЁӢдёӯгҖӮ

3 - HTTP зҡ„еҹәжң¬дјҳеҢ–

еҪұе“ҚдёҖдёӘHTTPзҪ‘з»ңиҜ·жұӮзҡ„еӣ зҙ дё»иҰҒжңүдёӨдёӘпјҡеёҰе®Ҫе’Ңе»¶иҝҹгҖӮ

еёҰе®Ҫпјҡ еҰӮжһңиҜҙжҲ‘д»¬иҝҳеҒңз•ҷеңЁжӢЁеҸ·дёҠзҪ‘зҡ„йҳ¶ж®өпјҢеёҰе®ҪеҸҜиғҪдјҡжҲҗдёәдёҖдёӘжҜ”иҫғдёҘйҮҚеҪұе“ҚиҜ·жұӮзҡ„й—®йўҳпјҢдҪҶжҳҜзҺ°еңЁзҪ‘з»ңеҹәзЎҖе»әи®ҫе·Із»ҸдҪҝеҫ—еёҰе®Ҫеҫ—еҲ°жһҒеӨ§зҡ„жҸҗеҚҮпјҢжҲ‘д»¬дёҚеҶҚдјҡжӢ…еҝғз”ұеёҰе®ҪиҖҢеҪұе“ҚзҪ‘йҖҹпјҢйӮЈд№Ҳе°ұеҸӘеү©дёӢе»¶иҝҹдәҶгҖӮ

е»¶иҝҹпјҡ

жөҸи§ҲеҷЁйҳ»еЎһпјҲHOL blockingпјүпјҡжөҸи§ҲеҷЁдјҡеӣ дёәдёҖдәӣеҺҹеӣ йҳ»еЎһиҜ·жұӮгҖӮжөҸи§ҲеҷЁеҜ№дәҺеҗҢдёҖдёӘеҹҹеҗҚпјҢеҗҢж—¶еҸӘиғҪжңү 4 дёӘиҝһжҺҘпјҲиҝҷдёӘж №жҚ®жөҸи§ҲеҷЁеҶ…ж ёдёҚеҗҢеҸҜиғҪдјҡжңүжүҖе·®ејӮпјүпјҢи¶…иҝҮжөҸи§ҲеҷЁжңҖеӨ§иҝһжҺҘж•°йҷҗеҲ¶пјҢеҗҺз»ӯиҜ·жұӮе°ұдјҡиў«йҳ»еЎһгҖӮ

DNS жҹҘиҜўпјҲDNS LookupпјүпјҡжөҸи§ҲеҷЁйңҖиҰҒзҹҘйҒ“зӣ®ж ҮжңҚеҠЎеҷЁзҡ„ IP жүҚиғҪе»әз«ӢиҝһжҺҘгҖӮе°ҶеҹҹеҗҚи§Јжһҗдёә IP зҡ„иҝҷдёӘзі»з»ҹе°ұжҳҜ DNSгҖӮиҝҷдёӘйҖҡеёёеҸҜд»ҘеҲ©з”ЁDNSзј“еӯҳз»“жһңжқҘиҫҫеҲ°еҮҸе°‘иҝҷдёӘж—¶й—ҙзҡ„зӣ®зҡ„гҖӮ

е»әз«ӢиҝһжҺҘпјҲInitial connectionпјүпјҡHTTP жҳҜеҹәдәҺ TCP еҚҸи®®зҡ„пјҢжөҸи§ҲеҷЁжңҖеҝ«д№ҹиҰҒеңЁз¬¬дёүж¬ЎжҸЎжүӢж—¶жүҚиғҪжҚҺеёҰ HTTP иҜ·жұӮжҠҘж–ҮпјҢиҫҫеҲ°зңҹжӯЈзҡ„е»әз«ӢиҝһжҺҘпјҢдҪҶжҳҜиҝҷдәӣиҝһжҺҘж— жі•еӨҚз”ЁдјҡеҜјиҮҙжҜҸж¬ЎиҜ·жұӮйғҪз»ҸеҺҶдёүж¬ЎжҸЎжүӢе’Ңж…ўеҗҜеҠЁгҖӮдёүж¬ЎжҸЎжүӢеңЁй«ҳе»¶иҝҹзҡ„еңәжҷҜдёӢеҪұе“ҚиҫғжҳҺжҳҫпјҢж…ўеҗҜеҠЁеҲҷеҜ№ж–Үд»¶зұ»еӨ§иҜ·жұӮеҪұе“ҚиҫғеӨ§гҖӮ

4 - HTTP1.0 е’Ң HTTP1.1 зҡ„дёҖдәӣеҢәеҲ«

HTTP1.0жңҖж—©еңЁзҪ‘йЎөдёӯдҪҝз”ЁжҳҜеңЁ1996е№ҙпјҢйӮЈдёӘж—¶еҖҷеҸӘжҳҜдҪҝз”ЁдёҖдәӣиҫғдёәз®ҖеҚ•зҡ„зҪ‘йЎөдёҠе’ҢзҪ‘з»ңиҜ·жұӮдёҠпјҢиҖҢHTTP1.1еҲҷеңЁ1999е№ҙжүҚејҖе§Ӣе№ҝжіӣеә”з”ЁдәҺзҺ°еңЁзҡ„еҗ„еӨ§жөҸи§ҲеҷЁзҪ‘з»ңиҜ·жұӮдёӯпјҢеҗҢж—¶HTTP1.1д№ҹжҳҜеҪ“еүҚдҪҝз”ЁжңҖдёәе№ҝжіӣзҡ„HTTPеҚҸи®®гҖӮ дё»иҰҒеҢәеҲ«дё»иҰҒдҪ“зҺ°еңЁпјҡ

зј“еӯҳеӨ„зҗҶ

еңЁHTTP1.0дёӯдё»иҰҒдҪҝз”ЁheaderйҮҢзҡ„If-Modified-Since,ExpiresжқҘеҒҡдёәзј“еӯҳеҲӨж–ӯзҡ„ж ҮеҮҶпјҢHTTP1.1еҲҷеј•е…ҘдәҶжӣҙеӨҡзҡ„зј“еӯҳжҺ§еҲ¶зӯ–з•ҘдҫӢеҰӮEntity tagпјҢIf-Unmodified-Since, If-Match, If-None-MatchзӯүжӣҙеӨҡеҸҜдҫӣйҖүжӢ©зҡ„зј“еӯҳеӨҙжқҘжҺ§еҲ¶зј“еӯҳзӯ–з•ҘгҖӮ

еёҰе®ҪдјҳеҢ–еҸҠзҪ‘з»ңиҝһжҺҘзҡ„дҪҝз”Ё

HTTP1.0дёӯпјҢеӯҳеңЁдёҖдәӣжөӘиҙ№еёҰе®Ҫзҡ„зҺ°иұЎпјҢдҫӢеҰӮе®ўжҲ·з«ҜеҸӘжҳҜйңҖиҰҒжҹҗдёӘеҜ№иұЎзҡ„дёҖйғЁеҲҶпјҢиҖҢжңҚеҠЎеҷЁеҚҙе°Ҷж•ҙдёӘеҜ№иұЎйҖҒиҝҮжқҘдәҶпјҢе№¶дё”дёҚж”ҜжҢҒж–ӯзӮ№з»ӯдј еҠҹиғҪпјҢHTTP1.1еҲҷеңЁиҜ·жұӮеӨҙеј•е…ҘдәҶrangeеӨҙеҹҹпјҢе®ғе…Ғи®ёеҸӘиҜ·жұӮиө„жәҗзҡ„жҹҗдёӘйғЁеҲҶпјҢеҚіиҝ”еӣһз ҒжҳҜ206пјҲPartial ContentпјүпјҢиҝҷж ·е°ұж–№дҫҝдәҶејҖеҸ‘иҖ…иҮӘз”ұзҡ„йҖүжӢ©д»ҘдҫҝдәҺе……еҲҶеҲ©з”ЁеёҰе®Ҫе’ҢиҝһжҺҘгҖӮ

й”ҷиҜҜйҖҡзҹҘзҡ„з®ЎзҗҶ

еңЁHTTP1.1дёӯж–°еўһдәҶ24дёӘй”ҷиҜҜзҠ¶жҖҒе“Қеә”з ҒпјҢеҰӮ409пјҲConflictпјүиЎЁзӨәиҜ·жұӮзҡ„иө„жәҗдёҺиө„жәҗзҡ„еҪ“еүҚзҠ¶жҖҒеҸ‘з”ҹеҶІзӘҒпјӣ410пјҲGoneпјүиЎЁзӨәжңҚеҠЎеҷЁдёҠзҡ„жҹҗдёӘиө„жәҗиў«ж°ёд№…жҖ§зҡ„еҲ йҷӨгҖӮ

HostеӨҙеӨ„зҗҶ

еңЁHTTP1.0дёӯи®ӨдёәжҜҸеҸ°жңҚеҠЎеҷЁйғҪз»‘е®ҡдёҖдёӘе”ҜдёҖзҡ„IPең°еқҖпјҢеӣ жӯӨпјҢиҜ·жұӮж¶ҲжҒҜдёӯзҡ„URLе№¶жІЎжңүдј йҖ’дё»жңәеҗҚпјҲhostnameпјүгҖӮдҪҶйҡҸзқҖиҷҡжӢҹдё»жңәжҠҖжңҜзҡ„еҸ‘еұ•пјҢеңЁдёҖеҸ°зү©зҗҶжңҚеҠЎеҷЁдёҠеҸҜд»ҘеӯҳеңЁеӨҡдёӘиҷҡжӢҹдё»жңәпјҲMulti-homed Web ServersпјүпјҢе№¶дё”е®ғд»¬е…ұдә«дёҖдёӘIPең°еқҖгҖӮHTTP1.1зҡ„иҜ·жұӮж¶ҲжҒҜе’Ңе“Қеә”ж¶ҲжҒҜйғҪеә”ж”ҜжҢҒHostеӨҙеҹҹпјҢдё”иҜ·жұӮж¶ҲжҒҜдёӯеҰӮжһңжІЎжңүHostеӨҙеҹҹдјҡжҠҘе‘ҠдёҖдёӘй”ҷиҜҜпјҲ400 Bad RequestпјүгҖӮ

й•ҝиҝһжҺҘ

HTTP 1.1ж”ҜжҢҒй•ҝиҝһжҺҘпјҲPersistentConnectionпјүе’ҢиҜ·жұӮзҡ„жөҒж°ҙзәҝпјҲPipeliningпјүеӨ„зҗҶпјҢеңЁдёҖдёӘTCPиҝһжҺҘдёҠеҸҜд»Ҙдј йҖҒеӨҡдёӘHTTPиҜ·жұӮе’Ңе“Қеә”пјҢеҮҸе°‘дәҶе»әз«Ӣе’Ңе…ій—ӯиҝһжҺҘзҡ„ж¶ҲиҖ—е’Ңе»¶иҝҹпјҢеңЁHTTP1.1дёӯй»ҳи®ӨејҖеҗҜConnectionпјҡ keep-aliveпјҢдёҖе®ҡзЁӢеәҰдёҠејҘиЎҘдәҶHTTP1.0жҜҸж¬ЎиҜ·жұӮйғҪиҰҒеҲӣе»әиҝһжҺҘзҡ„зјәзӮ№гҖӮд»ҘдёӢжҳҜеёёи§Ғзҡ„HTTP1.0пјҡ

еҢәеҲ«з”ЁдёҖеј еӣҫжқҘдҪ“зҺ°пјҡ

5 - HTTP1.0 е’Ң HTTP1.1 зҺ°еӯҳзҡ„дёҖдәӣй—®йўҳ

1) дёҠйқўжҸҗеҲ°иҝҮзҡ„пјҢHTTP1.xеңЁдј иҫ“ж•°жҚ®ж—¶пјҢжҜҸж¬ЎйғҪйңҖиҰҒйҮҚж–°е»әз«ӢиҝһжҺҘпјҢж— з–‘еўһеҠ дәҶеӨ§йҮҸзҡ„е»¶иҝҹж—¶й—ҙпјҢзү№еҲ«жҳҜеңЁз§»еҠЁз«ҜжӣҙдёәзӘҒеҮәгҖӮ

2) HTTP1.xеңЁдј иҫ“ж•°жҚ®ж—¶пјҢжүҖжңүдј иҫ“зҡ„еҶ…е®№йғҪжҳҜжҳҺж–ҮпјҢе®ўжҲ·з«Ҝе’ҢжңҚеҠЎеҷЁз«ҜйғҪж— жі•йӘҢиҜҒеҜ№ж–№зҡ„иә«д»ҪпјҢиҝҷеңЁдёҖе®ҡзЁӢеәҰдёҠж— жі•дҝқиҜҒж•°жҚ®зҡ„е®үе…ЁжҖ§гҖӮ

3) HTTP1.xеңЁдҪҝз”Ёж—¶пјҢheaderйҮҢжҗәеёҰзҡ„еҶ…е®№иҝҮеӨ§пјҢеңЁдёҖе®ҡзЁӢеәҰдёҠеўһеҠ дәҶдј иҫ“зҡ„жҲҗжң¬пјҢе№¶дё”жҜҸж¬ЎиҜ·жұӮheaderеҹәжң¬дёҚжҖҺд№ҲеҸҳеҢ–пјҢе°Өе…¶еңЁз§»еҠЁз«ҜеўһеҠ з”ЁжҲ·жөҒйҮҸгҖӮ

4) иҷҪз„¶HTTP1.xж”ҜжҢҒдәҶkeep-aliveпјҢжқҘејҘиЎҘеӨҡж¬ЎеҲӣе»әиҝһжҺҘдә§з”ҹзҡ„е»¶иҝҹпјҢдҪҶжҳҜkeep-aliveдҪҝз”ЁеӨҡдәҶеҗҢж ·дјҡз»ҷжңҚеҠЎз«ҜеёҰжқҘеӨ§йҮҸзҡ„жҖ§иғҪеҺӢеҠӣпјҢе№¶дё”еҜ№дәҺеҚ•дёӘж–Үд»¶иў«дёҚж–ӯиҜ·жұӮзҡ„жңҚеҠЎ(дҫӢеҰӮеӣҫзүҮеӯҳж”ҫзҪ‘з«ҷ)пјҢkeep-aliveеҸҜиғҪдјҡжһҒеӨ§зҡ„еҪұе“ҚжҖ§иғҪпјҢеӣ дёәе®ғеңЁж–Үд»¶иў«иҜ·жұӮд№ӢеҗҺиҝҳдҝқжҢҒдәҶдёҚеҝ…иҰҒзҡ„иҝһжҺҘеҫҲй•ҝж—¶й—ҙгҖӮ

6 - HTTPS еә”еЈ°иҖҢеҮә

дёәдәҶи§ЈеҶід»ҘдёҠй—®йўҳпјҢзҪ‘жҷҜеңЁ1994е№ҙеҲӣе»әдәҶHTTPSпјҢе№¶еә”з”ЁеңЁзҪ‘жҷҜеҜјиҲӘиҖ…жөҸи§ҲеҷЁдёӯгҖӮ жңҖеҲқпјҢHTTPSжҳҜдёҺSSLдёҖиө·дҪҝз”Ёзҡ„пјӣеңЁSSLйҖҗжёҗжј”еҸҳеҲ°TLSж—¶пјҲе…¶е®һдёӨдёӘжҳҜдёҖдёӘдёңиҘҝпјҢеҸӘжҳҜеҗҚеӯ—дёҚеҗҢиҖҢе·ІпјүпјҢжңҖж–°зҡ„HTTPSд№ҹз”ұеңЁ2000е№ҙдә”жңҲе…¬еёғзҡ„RFC 2818жӯЈејҸзЎ®е®ҡдёӢжқҘгҖӮз®ҖеҚ•жқҘиҜҙпјҢHTTPSе°ұжҳҜе®үе…ЁзүҲзҡ„HTTPпјҢе№¶дё”з”ұдәҺеҪ“д»Ҡж—¶д»ЈеҜ№е®үе…ЁжҖ§иҰҒжұӮжӣҙй«ҳпјҢchromeе’ҢfirefoxйғҪеӨ§еҠӣж”ҜжҢҒзҪ‘з«ҷдҪҝз”ЁHTTPSпјҢиӢ№жһңд№ҹеңЁios 10зі»з»ҹдёӯејәеҲ¶appдҪҝз”ЁHTTPSжқҘдј иҫ“ж•°жҚ®пјҢз”ұжӯӨеҸҜи§ҒHTTPSеҠҝеңЁеҝ…иЎҢгҖӮ

7 - HTTPS дёҺ HTTP зҡ„дёҖдәӣеҢәеҲ«

1) HTTPSеҚҸи®®йңҖиҰҒеҲ°CAз”іиҜ·иҜҒд№ҰпјҢдёҖиҲ¬е…Қиҙ№иҜҒд№ҰеҫҲе°‘пјҢйңҖиҰҒдәӨиҙ№гҖӮ

2) HTTPеҚҸи®®иҝҗиЎҢеңЁTCPд№ӢдёҠпјҢжүҖжңүдј иҫ“зҡ„еҶ…е®№йғҪжҳҜжҳҺж–ҮпјҢHTTPSиҝҗиЎҢеңЁSSL/TLSд№ӢдёҠпјҢSSL/TLSиҝҗиЎҢеңЁTCPд№ӢдёҠпјҢжүҖжңүдј иҫ“зҡ„еҶ…е®№йғҪз»ҸиҝҮеҠ еҜҶзҡ„гҖӮ

3) HTTPе’ҢHTTPSдҪҝз”Ёзҡ„жҳҜе®Ңе…ЁдёҚеҗҢзҡ„иҝһжҺҘж–№ејҸпјҢз”Ёзҡ„з«ҜеҸЈд№ҹдёҚдёҖж ·пјҢеүҚиҖ…жҳҜ80пјҢеҗҺиҖ…жҳҜ443гҖӮ

4) HTTPSеҸҜд»Ҙжңүж•Ҳзҡ„йҳІжӯўиҝҗиҗҘе•ҶеҠ«жҢҒпјҢи§ЈеҶідәҶйҳІеҠ«жҢҒзҡ„дёҖдёӘеӨ§й—®йўҳгҖӮ

8 - HTTPS ж”№йҖ 

еҰӮжһңдёҖдёӘзҪ‘з«ҷиҰҒе…Ёз«ҷз”ұHTTPжӣҝжҚўжҲҗHTTPSпјҢеҸҜиғҪйңҖиҰҒе…іжіЁд»ҘдёӢеҮ зӮ№пјҡ

1) е®үиЈ…CAиҜҒд№ҰпјҢдёҖиҲ¬зҡ„иҜҒд№ҰйғҪжҳҜйңҖиҰҒж”¶иҙ№зҡ„пјҢиҝҷиҫ№жҺЁиҚҗдёҖдёӘжҜ”иҫғеҘҪзҡ„иҙӯд№°иҜҒд№ҰзҪ‘з«ҷпјҡ1пјүLetвҖҷs EncryptпјҢе…Қиҙ№пјҢеҝ«жҚ·пјҢж”ҜжҢҒеӨҡеҹҹеҗҚпјҲдёҚжҳҜйҖҡй…Қз¬ҰпјүпјҢдёүжқЎе‘Ҫд»ӨеҚіж—¶зӯҫзҪІ+еҜјеҮәиҜҒд№ҰгҖӮзјәзӮ№жҳҜжҡӮж—¶еҸӘжңүдёүдёӘжңҲжңүж•ҲжңҹпјҢеҲ°жңҹйңҖз»ӯзӯҫгҖӮ2) Comodo PositiveSSLпјҢж”¶иҙ№пјҢдҪҶжҳҜжҜ”иҫғзЁіе®ҡгҖӮ

2) еңЁиҙӯд№°иҜҒд№Ұд№ӢеҗҺпјҢеңЁиҜҒд№ҰжҸҗдҫӣзҡ„зҪ‘з«ҷдёҠй…ҚзҪ®иҮӘе·ұзҡ„еҹҹеҗҚпјҢе°ҶиҜҒд№ҰдёӢиҪҪдёӢжқҘд№ӢеҗҺпјҢй…ҚзҪ®иҮӘе·ұзҡ„webжңҚеҠЎеҷЁпјҢеҗҢж—¶иҝӣиЎҢд»Јз Ғж”№йҖ гҖӮ

3) HTTPS йҷҚдҪҺз”ЁжҲ·и®ҝй—®йҖҹеәҰгҖӮSSLжҸЎжүӢпјҢHTTPS еҜ№йҖҹеәҰдјҡжңүдёҖе®ҡзЁӢеәҰзҡ„йҷҚдҪҺпјҢдҪҶжҳҜеҸӘиҰҒз»ҸиҝҮеҗҲзҗҶдјҳеҢ–е’ҢйғЁзҪІпјҢHTTPS еҜ№йҖҹеәҰзҡ„еҪұе“Қе®Ңе…ЁеҸҜд»ҘжҺҘеҸ—гҖӮеңЁеҫҲеӨҡеңәжҷҜдёӢпјҢHTTPS йҖҹеәҰе®Ңе…ЁдёҚйҖҠдәҺ HTTPпјҢеҰӮжһңдҪҝз”Ё SPDYпјҢHTTPS зҡ„йҖҹеәҰз”ҡиҮіиҝҳиҰҒжҜ” HTTP еҝ«гҖӮ

4) зӣёеҜ№дәҺHTTPSйҷҚдҪҺи®ҝй—®йҖҹеәҰпјҢе…¶е®һжӣҙйңҖиҰҒе…іеҝғзҡ„жҳҜжңҚеҠЎеҷЁз«Ҝзҡ„CPUеҺӢеҠӣпјҢHTTPSдёӯеӨ§йҮҸзҡ„еҜҶй’Ҙз®—жі•и®Ўз®—пјҢдјҡж¶ҲиҖ—еӨ§йҮҸзҡ„CPUиө„жәҗпјҢеҸӘжңүи¶іеӨҹзҡ„дјҳеҢ–пјҢHTTPS зҡ„жңәеҷЁжҲҗжң¬жүҚдёҚдјҡжҳҺжҳҫеўһеҠ гҖӮ

жҺЁиҚҗдёҖеҲҷж·ҳе®қзҪ‘ж”№йҖ HTTPSзҡ„ж–Үз« гҖӮ

9 - дҪҝз”ЁSPDYеҠ еҝ«дҪ зҡ„зҪ‘з«ҷйҖҹеәҰ

2012е№ҙgoogleеҰӮдёҖеЈ°жғҠйӣ·жҸҗеҮәдәҶSPDYзҡ„ж–№жЎҲпјҢеӨ§е®¶жүҚејҖе§Ӣд»ҺжӯЈйқўзңӢеҫ…е’Ңи§ЈеҶіиҖҒзүҲжң¬HTTPеҚҸи®®жң¬иә«зҡ„й—®йўҳпјҢSPDYеҸҜд»ҘиҜҙжҳҜз»јеҗҲдәҶHTTPSе’ҢHTTPдёӨиҖ…жңүзӮ№дәҺдёҖдҪ“зҡ„дј иҫ“еҚҸи®®пјҢдё»иҰҒи§ЈеҶіпјҡ

йҷҚдҪҺе»¶иҝҹ

й’ҲеҜ№HTTPй«ҳе»¶иҝҹзҡ„й—®йўҳпјҢSPDYдјҳйӣ…зҡ„йҮҮеҸ–дәҶеӨҡи·ҜеӨҚз”ЁпјҲmultiplexingпјүгҖӮеӨҡи·ҜеӨҚз”ЁйҖҡиҝҮеӨҡдёӘиҜ·жұӮstreamе…ұдә«дёҖдёӘtcpиҝһжҺҘзҡ„ж–№ејҸпјҢи§ЈеҶідәҶHOL blockingзҡ„й—®йўҳпјҢйҷҚдҪҺдәҶе»¶иҝҹеҗҢж—¶жҸҗй«ҳдәҶеёҰе®Ҫзҡ„еҲ©з”ЁзҺҮгҖӮ

иҜ·жұӮдјҳе…Ҳзә§пјҲrequest prioritizationпјү

еӨҡи·ҜеӨҚз”ЁеёҰжқҘдёҖдёӘж–°зҡ„й—®йўҳжҳҜпјҢеңЁиҝһжҺҘе…ұдә«зҡ„еҹәзЎҖд№ӢдёҠжңүеҸҜиғҪдјҡеҜјиҮҙе…ій”®иҜ·жұӮиў«йҳ»еЎһгҖӮSPDYе…Ғи®ёз»ҷжҜҸдёӘrequestи®ҫзҪ®дјҳе…Ҳзә§пјҢиҝҷж ·йҮҚиҰҒзҡ„иҜ·жұӮе°ұдјҡдјҳе…Ҳеҫ—еҲ°е“Қеә”гҖӮжҜ”еҰӮжөҸи§ҲеҷЁеҠ иҪҪйҰ–йЎөпјҢйҰ–йЎөзҡ„htmlеҶ…е®№еә”иҜҘдјҳе…Ҳеұ•зӨәпјҢд№ӢеҗҺжүҚжҳҜеҗ„з§ҚйқҷжҖҒиө„жәҗж–Үд»¶пјҢи„ҡжң¬ж–Үд»¶зӯүеҠ иҪҪпјҢиҝҷж ·еҸҜд»ҘдҝқиҜҒз”ЁжҲ·иғҪз¬¬дёҖж—¶й—ҙзңӢеҲ°зҪ‘йЎөеҶ…е®№гҖӮ

headerеҺӢзј©

еүҚйқўжҸҗеҲ°HTTP1.xзҡ„headerеҫҲеӨҡж—¶еҖҷйғҪжҳҜйҮҚеӨҚеӨҡдҪҷзҡ„гҖӮйҖүжӢ©еҗҲйҖӮзҡ„еҺӢзј©з®—жі•еҸҜд»ҘеҮҸе°ҸеҢ…зҡ„еӨ§е°Ҹе’Ңж•°йҮҸгҖӮ

еҹәдәҺHTTPSзҡ„еҠ еҜҶеҚҸи®®дј иҫ“

еӨ§еӨ§жҸҗй«ҳдәҶдј иҫ“ж•°жҚ®зҡ„еҸҜйқ жҖ§гҖӮ

жңҚеҠЎз«ҜжҺЁйҖҒпјҲserver pushпјү

йҮҮз”ЁдәҶSPDYзҡ„зҪ‘йЎөпјҢдҫӢеҰӮжҲ‘зҡ„зҪ‘йЎөжңүдёҖдёӘsytle.cssзҡ„иҜ·жұӮпјҢеңЁе®ўжҲ·з«Ҝж”¶еҲ°sytle.cssж•°жҚ®зҡ„еҗҢж—¶пјҢжңҚеҠЎз«Ҝдјҡе°Ҷsytle.jsзҡ„ж–Үд»¶жҺЁйҖҒз»ҷе®ўжҲ·з«ҜпјҢеҪ“е®ўжҲ·з«ҜеҶҚж¬Ўе°қиҜ•иҺ·еҸ–sytle.jsж—¶е°ұеҸҜд»ҘзӣҙжҺҘд»Һзј“еӯҳдёӯиҺ·еҸ–еҲ°пјҢдёҚз”ЁеҶҚеҸ‘иҜ·жұӮдәҶгҖӮSPDYжһ„жҲҗеӣҫпјҡ

SPDYдҪҚдәҺHTTPд№ӢдёӢпјҢTCPе’ҢSSLд№ӢдёҠпјҢиҝҷж ·еҸҜд»ҘиҪ»жқҫе…је®№иҖҒзүҲжң¬зҡ„HTTPеҚҸи®®(е°ҶHTTP1.xзҡ„еҶ…е®№е°ҒиЈ…жҲҗдёҖз§Қж–°зҡ„frameж јејҸ)пјҢеҗҢж—¶еҸҜд»ҘдҪҝз”Ёе·Іжңүзҡ„SSLеҠҹиғҪгҖӮ

е…је®№жҖ§пјҡ

10 - HTTP2.0 зҡ„еүҚдё–д»Ҡз”ҹ

йЎҫеҗҚжҖқд№үжңүдәҶHTTP1.xпјҢйӮЈд№ҲHTTP2.0д№ҹе°ұйЎәзҗҶжҲҗз« зҡ„еҮәзҺ°дәҶгҖӮHTTP2.0еҸҜд»ҘиҜҙжҳҜSPDYзҡ„еҚҮзә§зүҲпјҲе…¶е®һеҺҹжң¬д№ҹжҳҜеҹәдәҺSPDYи®ҫи®Ўзҡ„пјүпјҢдҪҶжҳҜпјҢHTTP2.0 и·ҹ SPDY д»ҚжңүдёҚеҗҢзҡ„ең°ж–№пјҢдё»иҰҒжҳҜд»ҘдёӢдёӨзӮ№пјҡ

HTTP2.0 ж”ҜжҢҒжҳҺж–Ү HTTP дј иҫ“пјҢиҖҢ SPDY ејәеҲ¶дҪҝз”Ё HTTPS

HTTP2.0 ж¶ҲжҒҜеӨҙзҡ„еҺӢзј©з®—жі•йҮҮз”Ё HPACKпјҢиҖҢйқһ SPDY йҮҮз”Ёзҡ„ DEFLATE

11 - HTTP2.0 зҡ„ж–°зү№жҖ§

ж–°зҡ„дәҢиҝӣеҲ¶ж јејҸпјҲBinary Formatпјү

HTTP1.xзҡ„и§ЈжһҗжҳҜеҹәдәҺж–Үжң¬гҖӮеҹәдәҺж–Үжң¬еҚҸи®®зҡ„ж јејҸи§ЈжһҗеӯҳеңЁеӨ©з„¶зјәйҷ·пјҢж–Үжң¬зҡ„иЎЁзҺ°еҪўејҸжңүеӨҡж ·жҖ§пјҢиҰҒеҒҡеҲ°еҒҘеЈ®жҖ§иҖғиҷ‘зҡ„еңәжҷҜеҝ…з„¶еҫҲеӨҡпјҢдәҢиҝӣеҲ¶еҲҷдёҚеҗҢпјҢеҸӘи®Ө0е’Ң1зҡ„з»„еҗҲгҖӮеҹәдәҺиҝҷз§ҚиҖғиҷ‘HTTP2.0зҡ„еҚҸи®®и§ЈжһҗеҶіе®ҡйҮҮз”ЁдәҢиҝӣеҲ¶ж јејҸпјҢе®һзҺ°ж–№дҫҝдё”еҒҘеЈ®гҖӮ

еӨҡи·ҜеӨҚз”ЁпјҲMultiPlexingпјү

еҚіиҝһжҺҘе…ұдә«пјҢеҚіжҜҸдёҖдёӘrequestйғҪжҳҜжҳҜз”ЁдҪңиҝһжҺҘе…ұдә«жңәеҲ¶зҡ„гҖӮдёҖдёӘrequestеҜ№еә”дёҖдёӘidпјҢиҝҷж ·дёҖдёӘиҝһжҺҘдёҠеҸҜд»ҘжңүеӨҡдёӘrequestпјҢжҜҸдёӘиҝһжҺҘзҡ„requestеҸҜд»ҘйҡҸжңәзҡ„ж··жқӮеңЁдёҖиө·пјҢжҺҘж”¶ж–№еҸҜд»Ҙж №жҚ®requestзҡ„ idе°ҶrequestеҶҚеҪ’еұһеҲ°еҗ„иҮӘдёҚеҗҢзҡ„жңҚеҠЎз«ҜиҜ·жұӮйҮҢйқўгҖӮеӨҡи·ҜеӨҚз”ЁеҺҹзҗҶеӣҫпјҡ

headerеҺӢзј©

еҰӮдёҠж–ҮдёӯжүҖиЁҖпјҢеҜ№еүҚйқўжҸҗеҲ°иҝҮHTTP1.xзҡ„headerеёҰжңүеӨ§йҮҸдҝЎжҒҜпјҢиҖҢдё”жҜҸж¬ЎйғҪиҰҒйҮҚеӨҚеҸ‘йҖҒпјҢHTTP2.0дҪҝз”ЁencoderжқҘеҮҸе°‘йңҖиҰҒдј иҫ“зҡ„headerеӨ§е°ҸпјҢйҖҡи®ҜеҸҢж–№еҗ„иҮӘcacheдёҖд»Ҫheader fieldsиЎЁпјҢж—ўйҒҝе…ҚдәҶйҮҚеӨҚheaderзҡ„дј иҫ“пјҢеҸҲеҮҸе°ҸдәҶйңҖиҰҒдј иҫ“зҡ„еӨ§е°ҸгҖӮ

жңҚеҠЎз«ҜжҺЁйҖҒпјҲserver pushпјү

еҗҢSPDYдёҖж ·пјҢHTTP2.0д№ҹе…·жңүserver pushеҠҹиғҪгҖӮзӣ®еүҚпјҢжңүеӨ§еӨҡж•°зҪ‘з«ҷе·Із»ҸеҗҜз”ЁHTTP2.0пјҢдҫӢеҰӮYouTuBeпјҢж·ҳе®қзҪ‘зӯүзҪ‘з«ҷпјҢеҲ©з”ЁchromeжҺ§еҲ¶еҸ°еҸҜд»ҘжҹҘзңӢжҳҜеҗҰеҗҜз”ЁH2пјҡ

жӣҙеӨҡе…ідәҺHTTP2зҡ„й—®йўҳеҸҜд»ҘеҸӮиҖғпјҡHTTP2еҘҮеҰҷж—ҘеёёпјҢд»ҘеҸҠHTTP2.0зҡ„е®ҳж–№зҪ‘з«ҷгҖӮ

е…ідәҺHTTP2е’ҢHTTP1.xзҡ„еҢәеҲ«еӨ§иҮҙеҸҜд»ҘзңӢдёӢеӣҫпјҡ

12 - HTTP2.0 зҡ„еҚҮзә§ж”№йҖ 

еҜ№жҜ”HTTPSзҡ„еҚҮзә§ж”№йҖ пјҢHTTP2.0жҲ–и®ёдјҡзЁҚеҫ®з®ҖеҚ•дёҖдәӣпјҢдҪ еҸҜиғҪйңҖиҰҒе…іжіЁд»ҘдёӢй—®йўҳпјҡ

1) еүҚж–ҮиҜҙдәҶHTTP2.0е…¶е®һеҸҜд»Ҙж”ҜжҢҒйқһHTTPSзҡ„пјҢдҪҶжҳҜзҺ°еңЁдё»жөҒзҡ„жөҸи§ҲеҷЁеғҸchromeпјҢfirefoxиЎЁзӨәиҝҳжҳҜеҸӘж”ҜжҢҒеҹәдәҺ TLS йғЁзҪІзҡ„HTTP2.0еҚҸи®®пјҢжүҖд»ҘиҰҒжғіеҚҮзә§жҲҗHTTP2.0иҝҳжҳҜе…ҲеҚҮзә§HTTPSдёәеҘҪгҖӮ

2) еҪ“дҪ зҡ„зҪ‘з«ҷе·Із»ҸеҚҮзә§HTTPSд№ӢеҗҺпјҢйӮЈд№ҲеҚҮзә§HTTP2.0е°ұз®ҖеҚ•еҫҲеӨҡпјҢеҰӮжһңдҪ дҪҝз”ЁNGINXпјҢеҸӘиҰҒеңЁй…ҚзҪ®ж–Үд»¶дёӯеҗҜеҠЁзӣёеә”зҡ„еҚҸи®®е°ұеҸҜд»ҘдәҶпјҢеҸҜд»ҘеҸӮиҖғNGINXзҷҪзҡ®д№ҰпјҢNGINXй…ҚзҪ®HTTP2.0е®ҳж–№жҢҮеҚ—гҖӮ

3) дҪҝз”ЁдәҶHTTP2.0йӮЈд№ҲпјҢеҺҹжң¬зҡ„HTTP1.xжҖҺд№ҲеҠһпјҢиҝҷдёӘй—®йўҳе…¶е®һдёҚз”ЁжӢ…еҝғпјҢHTTP2.0е®Ңе…Ёе…је®№HTTP1.xзҡ„иҜӯд№үпјҢеҜ№дәҺдёҚж”ҜжҢҒHTTP2.0зҡ„жөҸи§ҲеҷЁпјҢNGINXдјҡиҮӘеҠЁеҗ‘дёӢе…је®№зҡ„гҖӮ
